# UX · 모션 아이디어 (통통 튀고 귀엽게)

## 원칙
"통통 튀는 귀여움"은 기능 목록이 아니라 **일관된 모션·손맛 언어**다. 한 번 시스템으로 정의하면 앱 전체에 싸게 퍼진다. 손으로 일일이 애니메이션하지 말 것.

## 모션 디자인 시스템 (먼저 정의)
- **스프링 프리셋 2~3개**: 말랑 / 통통 / 탄력
- **햅틱 + 사운드 팔레트 5~6개**: 전송 · 수신 · 좋아요 · 레벨업 · 에러
- **Squash & Stretch 규칙**: 등장·전송·터치에 공통 적용
- Flutter 도구: `flutter_animate`, `SpringSimulation`, Rive(캐릭터 외 벡터 효과), `HapticFeedback`, `audioplayers`

## 아이디어 카탈로그

### 1. 통통 튀는 모션
- 말풍선 스프링 오버슈트 등장 (페이드 X → 톡! 튀어 들어옴)
- Squash & Stretch: 보낼 때 늘었다 안착 / 받을 때 아래서 통 튀어오름
- 전송 버튼 눌림(squash) → 스프링 복귀, 메시지 발사감
- 말풍선 long-press 시 젤리처럼 늘어나며 리액션 메뉴

### 2. 어디서나 살아있는 캐릭터
- idle 마이크로모션: 숨쉬기 · 눈 깜빡 · 가끔 두리번 (정지 = 죽은 앱)
- **입력 반응**: 타이핑 시작하면 캐릭터가 쫑긋 고개 들고 쳐다봄 (저비용·고임팩트)
- 터치 반응(탭/연타 → 까르르/어지러움), 스크롤 추적

### 3. 손맛: 햅틱 + 사운드 (비밀병기)
- 전송/수신 햅틱 톡톡, 스티커엔 통통
- 사운드 팔레트 5~6개 (도착 "뽁", 좋아요 "뾰롱", 레벨업 팡파레)
- ⚠️ 이게 빠지면 모션이 좋아도 "화면 속 그림" — 손맛이 "살아있음"을 완성

### 4. 보상의 순간
- 호감도 레벨업: 하트팝 + confetti + 캐릭터 점프
- 첫 만남 연출(알 깨기 / 문 빼꼼) = 온보딩을 이벤트로
- 연속 접속(streak) 배지·스티커 언락

### 5. 살아있는 배경 (Ambient)
- 시간대 반응(아침 환함 / 밤 어둑 + 캐릭터 졸림), 날씨·계절, 시즌 의상
- 풀 투 리프레시 시 캐릭터가 끌려왔다 "왜 당겨~" 하고 튕김

### 6. 꾸미기 자유도
- 말풍선 스킨 · 배경 테마 · 폰트 · 캐릭터 옷·액세서리
- 호감도로 언락 → **꾸미기 + 진행을 결합**해 애착 ↑

### 7. 이스터에그
- 폰 흔들기 → 어지러움, 볼 꼬집기/쓰다듬기 제스처, 특정 단어에 숨은 반응

## 프로토타입 "통통 4종" (임팩트 최고 / 비용 최저)
1. 말풍선 스프링 등장 + 전송 버튼 squash
2. 캐릭터 idle 숨쉬기·깜빡 + **타이핑하면 쳐다보기**
3. 전송/수신 햅틱 + 사운드 2~3개
4. 호감도 레벨업 하트팝 1개

→ 이 4개면 첫 데모에서 "어 얘 살아있네?"가 바로 나온다.
