# 노아의 방 — 가구 스프라이트 스펙

아이소메트릭 방(파니룸 스타일)에 꽂을 **가구 PNG** 규격. 아래대로 만들어 이 폴더에 넣으면
`kUseFurnitureSprites = true`(app/lib/room/furniture.dart) + pubspec 등록만으로 바로 적용됩니다.

## 필요한 파일 (9개, 파일명 고정 — 소문자)
| 파일 | 가구 | 대략 폭(타일 기준) |
|---|---|---|
| `bed.png` | 침대(이불·베개) | 2.0 |
| `desk.png` | 책상(+의자/소품) | 1.5 |
| `sofa.png` | 소파(쿠션·팔걸이) | 2.0 |
| `lamp.png` | 플로어 스탠드 | 0.9 |
| `plant.png` | 화분 식물 | 1.0 |
| `fridge.png` | 냉장고 | 1.3 |
| `counter.png` | 싱크대(주방 하부장+싱크) | 2.2 |
| `table.png` | 커피테이블(+소품) | 1.4 |
| `rug.png` | 러그(바닥, 높이 0) | 2.0 |

## 공통 규격
- **투영**: 2:1 아이소메트릭(가로:세로 = 2:1, 약 30° 부감). 모든 가구 **같은 각도**.
- **배경**: 완전 투명(알파). 그림자는 넣지 말 것(앱이 바닥 그림자를 따로 그림).
- **앵커**: 가구가 바닥에 닿는 점(footprint 중앙)이 **이미지 가로 중앙 + 하단 가까이**에 오게. 좌우 여백 최소.
- **해상도**: 정사각 캔버스 권장(예: 512×512), 가구가 캔버스를 꽉 채우되 위 약간 여백. @2x로 선명하게.
- **스타일**: 둥근 카툰풍 + **진한 외곽선**(방의 외곽선 룩과 일치) + 부드러운 셀 음영. 광원 **좌상단**.
- **팔레트(민트 방과 조화)**: 침대=연하늘, 소파=핑크, 책상/테이블/스탠드=따뜻한 우드, 냉장고=크림화이트,
  싱크대=크림+스틸 싱크, 화분=테라코타+초록, 러그=옐로우 계열. (현재 코드 색감 참고)
- **일관성**: 9개 모두 같은 화가/모델/세팅으로 한 번에 — 선 두께·음영·채도 통일이 제일 중요.

## 적용 절차 (아트 넣은 뒤)
1. PNG 9개를 이 폴더(`app/assets/room/`)에 넣는다.
2. `app/pubspec.yaml` 의 `flutter:` 아래에 추가:
   ```yaml
   assets:
     - assets/room/
   ```
3. `app/lib/room/furniture.dart` 의 `kUseFurnitureSprites` 를 `true` 로.
4. Claude에게 "에셋 넣었어" 라고 하면 폭 배수·앵커 미세조정 후 스크린샷으로 확인.

## 아트 소스 옵션
- **에셋 팩 구매/다운로드**: itch.io·Kenney 등 "isometric furniture / room" 팩 (스타일 일치 확인 필요).
- **이미지 생성 모델**: `PROMPT.txt` 의 프롬프트로 9종 일괄 생성(시드·스타일 고정).
- **직접 제작**: 위 규격대로.
